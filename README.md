## Instruções

  - Instalar as bibliotecas encontradas em /lib
  - Rodar o script 'database_script.sql' na pasta root do projeto
  - Rodar a build do projeto
  - Executar

  - MySQL 8.0
  - JDK 1.8
  - JavaFX 8.0.111-b14
  - asm-3.2
  - groovy-1.7.2
  - iText-2.1.7
  - jasperreports-6.3.1
  - jasperreports-fonts-6.3.1
  - jasperreports-javaflow-6.3.1
